## Insolvency Insight: Corporate Bankruptcy Prediction Using Machine Learning

# Overview

InsolvencyInsight is an end-to-end Machine Learning project designed to predict corporate bankruptcy using financial indicators from Taiwanese companies. The project applies data preprocessing, exploratory data analysis, feature engineering, class imbalance handling, and multiple classification models to identify companies at risk of financial distress.

The objective is to assist investors, financial analysts, lenders, and stakeholders in making informed decisions through data-driven bankruptcy risk assessment.

---

# Dataset

Dataset: Taiwanese Bankruptcy Prediction Dataset

- Total Companies: 3,753
- Features: 95 Financial Indicators
- Target Variable:
  - "0" = Non-Bankrupt Company
  - "1" = Bankrupt Company

The dataset contains financial ratios related to profitability, liquidity, leverage, operational efficiency, and cash flow.

---

# Project Workflow

1. Data Cleaning

- Missing value detection and treatment
- Dataset validation
- Target variable analysis

2. Exploratory Data Analysis (EDA)

- Correlation analysis
- Feature relationship visualization
- Distribution analysis
- Heatmap generation

3. Data Preprocessing

- Train-Test Split
- Standardization using StandardScaler
- Class imbalance handling using SMOTE

4. Machine Learning Models

- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier

5. Model Evaluation

The following metrics were used:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

---

# Model Performance
| Model | Accuracy | Precision | Recall | F1 Score | ROC-AUC |
|-------|----------|-----------|--------|----------|---------|
| Logistic Regression | 89.33% | 31.48% | 85.00% | 45.95% | 82.90% |
| Random Forest | 93.87% | 44.00% | 55.00% | 48.89% | 94.31% |
| XGBoost | *94.40%* | *48.39%* | *75.00%* | *58.82%* | *95.94%* |

---

# Final Model

Selected Model: XGBoost

Performance

- Accuracy: 94.40%
- Precision: 48.39%
- Recall: 75.00%
- F1 Score: 58.82%
- ROC-AUC: 95.94%

XGBoost demonstrated the strongest overall performance and was selected as the final deployment model.

# Key Financial Indicators

The most influential variables identified by the model include:

- Borrowing Dependency
- Debt Ratio %
- Liability to Equity
- Net Worth / Assets
- Net Income to Total Assets
- Non-industry Income and Expenditure / Revenue

These indicators highlight the importance of leverage, profitability, and financial stability in predicting corporate distress.
